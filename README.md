Appility!
===================


Appility es un catalogo conformado por una lista de aplicaciones, categorizado que muestra su respectivo logos y al hacer [tap] muestra informacion en detalle.

----------


Requerimientos
-------------

 - iOS + 9 
 - Swift 2.3 
 - Xcode +8

Librerias
---------

 - SwiftyJSON
